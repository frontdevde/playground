# React / Flummox demo app

Small demo application using React and Flummox to render/filter technology lists.
Based on [rendro](https://github.com/rendro/)'s great React example.


## Initial installation

```
$ npm install
$ npm install -g jspm
$ jspm install
```

## Start JSON Server

```
$ npm run api
```

## Start dev server

```
$ npm start
```
